# dotfiles

## Setup
1. Install GNU stow
2. Clone repo 
3. cd `dotfiles`
4. To setup neovim, run `./scripts/linux-nvim`
5. To setup neovim dotfiels, run `stow nvim`
6. Run neovim to automatically install the plugins on first run
