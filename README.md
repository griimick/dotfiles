# dotfiles

repo includes personal dotfiles.

```bash

	cd dotfiles

	stow default

```

## Uses 

1. bash
2. stow
3. vim 8.2+ (floating window support)
4. nvim (flavoured vim)
5. ripgrep (flavoured grep)
6. bat (flavoured cat)
7. fd (flavoured find)
