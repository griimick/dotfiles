## Uses 

1. bash
2. stow
3. vim 8.2+ (floating window support)
4. nvim (flavoured vim)

## Color schemes

1. [gruvbox](https://github.com/gruvbox-community/gruvbox)
2. [catppuccin](https://github.com/catppuccin/catppuccin)

## Experimental

1. ripgrep (flavoured grep)
2. bat (flavoured cat)
3. fd (flavoured find)
